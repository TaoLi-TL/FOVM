## **Installation & Usage**

### **Prerequisites**
- MATLAB (Tested on R2021a or later)

### **Run the Demo**
1. Place your hazy nighttime images in the `input/` folder.
2. Run the `demo.m` script:
   ```matlab
   demo
   ```
3. The processed images will be saved in the `output/` folder.
